### ID/MD设计 ###

1. ID/MD设计人员？外包公司？
2. 2B/2C产品？
3. 天马行空思功能，简约去繁出设计
4. ID设计 
	-  ID设计符合主流大众审美观
	-  考虑MD实现可行性（DFM，包括MD设计、加工实现、加工成本等因素）
	-  考虑工艺可行性
	-  需MD设计师一同评估（或者有经验的HW-PM）
5. MD设计
	- 注意加工成本
		- 合理拆件
		- 减少拆件风险
		- 考虑组装成本
	- 防水防尘设计
	- 细节美感与手感
	- 可以跟模，有相关经验
	- 与包装设计协同，注意包装美观、抗摔、有逼格
